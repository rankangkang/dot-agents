# agent-spec

AI 资产管理仓库 — 统一存放和管理 Agent 相关的技能（Skills）、规则（Rules）、脚本（Scripts）、命令（Commands）等配置资产。

## 快速开始

所有 AI 资产统一存放在 `.agents/` 目录下，详见 [.agents/README.md](.agents/README.md)。

## 目录结构

```
.
├── .agents/
│   ├── skills/      # Agent 技能定义
│   ├── rules/       # Agent 行为规则
│   ├── scripts/     # 自动化脚本
│   ├── commands/    # 自定义命令
│   ├── prompts/     # Prompt 模板
│   └── templates/   # 项目/文件模板
└── README.md
```
